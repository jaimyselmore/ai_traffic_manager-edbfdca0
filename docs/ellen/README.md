# Ellen - AI Planning Agent

Ellen is de AI-gebaseerde planning agent voor het AI Traffic Manager systeem. Ellen helpt met het automatiseren van planning, capaciteitsanalyse, en het optimaliseren van project workflows.

## Overzicht

### Wat doet Ellen?

Ellen is een conversationele AI agent die:
- **Planning automatiseert** - Projecten en meetings inplannen op basis van beschikbare capaciteit
- **Conflicten detecteert** - Overlappingen en capaciteitsproblemen signaleert
- **Suggesties doet** - Optimale planning voorstelt gebaseerd op constraints
- **Context begrijpt** - Project titels, team capaciteit, en deadlines analyseert
- **Interactief werkt** - Via chat interface met gebruikers communiceert

### Kernfunctionaliteit

```
1. PROJECT PLANNING
   - Nieuwe projecten automatisch inplannen
   - Team toewijzen op basis van discipline en capaciteit
   - Fases distribueren over beschikbare tijd

2. MEETING SCHEDULING
   - Gemeenschappelijke tijdslots vinden
   - Deelnemers automatisch notificeren
   - Conflicten vermijden

3. CAPACITY ANALYSIS
   - Huidige workload per medewerker inzien
   - Overbelaste periodes detecteren
   - Herverdeling suggereren

4. CHANGE MANAGEMENT
   - Wijzigingen aan lopende projecten verwerken
   - Impact analyse op andere projecten
   - Alternatieve planning voorstellen
